# キャッシュアウトコントロール データ分析基盤 - 設計ドキュメント

## 📋 プロジェクト概要

旅行業における国内事業のサプライヤー支払管理を対象とした、キャッシュアウトコントロールのためのデータ分析基盤の設計ドキュメント一式です。

### 目的
- 支払タイミングの最適化によるキャッシュフロー改善
- 支払予測精度の向上
- チャネル別・サプライヤー種別の多角的分析

### 対象範囲
- **ビジネスチャネル**: 顧客予約型、事前仕入型、月次一括精算型
- **サプライヤー種別**: 宿泊施設、交通機関（航空・鉄道・バス）、観光施設、飲食店
- **データソース**: 予約管理システム、会計システム、支払管理システム

## 🎯 主要KPI一覧

### キャッシュフロー管理
1. **KPI-001**: 日次支払予測額
2. **KPI-002**: 週次キャッシュアウト実績
3. **KPI-003**: 月次キャッシュアウト予算達成率

### 支払予測精度
4. **KPI-004**: 支払予測精度（MAPE）
5. **KPI-005**: 支払遅延率

### サプライヤー管理
6. **KPI-006**: サプライヤー別支払集中度
7. **KPI-007**: サプライヤー別平均支払サイト

### チャネル別分析
8. **KPI-008**: チャネル別支払構成比
9. **KPI-009**: チャネル別平均支払リードタイム

### 異常検知
10. **KPI-010**: 支払額異常検知率

## 🗂️ データモデル概要

### 主要エンティティ

```
予約（Reservation）
  ↓ 1:N
予約明細（Reservation Detail）
  ↓ 1:N
仕入明細（Procurement Detail）
  ↓ N:1
請求（Invoice）
  ↓ 1:N
支払（Payment）
```

### マスタデータ
- サプライヤー（Supplier）
- サプライヤー種別（Supplier Type）
- チャネル（Channel）
- サービス種別（Service Type）
- 支払条件（Payment Terms）
- 支払方法（Payment Method）
- 支払ステータス（Payment Status）
- 会計期間（Fiscal Period）

## 🎨 ダッシュボード構成

1. **キャッシュフロー予測ダッシュボード**
   - 30日間支払予測カレンダー
   - 支払予測トレンドグラフ
   - サプライヤー種別別予測

2. **実績管理ダッシュボード**
   - 週次支払実績トレンド
   - 月次予算達成率
   - KPIサマリー

3. **予測精度ダッシュボード**
   - MAPE推移グラフ
   - 予測期間別の精度比較

4. **サプライヤー分析ダッシュボード**
   - 上位サプライヤーランキング
   - パレート図
   - 支払サイト分析

5. **チャネル分析ダッシュボード**
   - チャネル別支払構成比
   - リードタイム比較

6. **異常検知ダッシュボード**
   - 異常検知アラート一覧
   - 異常パターン分析
