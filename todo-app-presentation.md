# ToDoアプリ プレゼンテーション資料

## 📋 アプリケーション概要

**シンプルで使いやすいタスク管理アプリケーション**

チーム全体のタスクを一元管理し、効率的なプロジェクト進行をサポートします。

---

## ✨ 主要機能

### 1. 📊 タスク一覧表示
- **視覚的なタスク管理**
  - すべてのタスクを一目で確認
  - 完了/未完了の状態を即座に把握
  - リアルタイムで統計情報を表示（全タスク数、未完了数、完了数）

- **スマートなソート機能**
  - 優先度順に自動整列（高 → 中 → 低）
  - 期限が近いタスクを優先表示
  - 完了タスクは最後に表示

### 2. ➕ タスク登録機能
- **充実した入力項目**
  - タスク名（必須）
  - 詳細説明（任意、最大500文字）
  - 期限設定（カレンダー選択）
  - 優先度選択（高/中/低）
  - **担当者名**（任意、最大50文字）

- **バリデーション機能**
  - 必須項目のチェック
  - 文字数制限
  - 過去の日付は選択不可

### 3. ✏️ タスク編集・詳細表示
- **柔軟な編集機能**
  - すべての項目を編集可能
  - 完了状態の切り替え
  - 作成日時の自動記録・表示

- **タスク削除機能**
  - 確認ダイアログで誤削除を防止

### 4. ✅ ワンクリック完了切り替え
- タスク一覧画面から直接チェックボックスで完了/未完了を切り替え
- 即座に画面に反映

### 5. 👤 担当者管理
- 各タスクに担当者を割り当て
- 一覧画面で担当者を一目で確認
- チーム全体のタスク分担を可視化

### 6. 💾 データ永続化
- ブラウザのLocalStorageに自動保存
- ページをリロードしてもデータが保持される
- インターネット接続不要で動作

## 🎯 優先度管理

### 3段階の優先度設定
- **高優先度**（赤色バッジ）
  - 緊急性の高いタスク
  - 一覧の最上部に表示

- **中優先度**（黄色バッジ）
  - 通常のタスク
  - デフォルト設定

- **低優先度**（緑色バッジ）
  - 時間のあるときに対応するタスク

### 視覚的な識別
- カラーバッジで優先度を即座に判別
- 重要なタスクを見逃さない

---

## 📅 期限管理

### 期限設定機能
- カレンダーUIで簡単に日付選択
- 過去の日付は選択不可（バリデーション）
- 期限が近いタスクを優先表示

### 期限表示
- 日本語形式で見やすく表示
- 一覧画面で期限を一目で確認

---

## 📈 統計情報ダッシュボード

### リアルタイム統計
- **全タスク数**: 登録されているすべてのタスク
- **未完了数**: 対応が必要なタスク
- **完了数**: 完了したタスク

### 進捗の可視化
- 数値で進捗状況を把握
- チームの生産性を測定

---

## 🔄 画面遷移フロー

```
[タスク一覧画面]
    ↓
    ├─→ [+ボタン] → [タスク登録画面] → [保存] → [タスク一覧画面]
    │
    └─→ [タスク選択] → [タスク詳細・編集画面] → [保存/削除] → [タスク一覧画面]
```

### シンプルな3画面構成
1. **タスク一覧画面**: メイン画面
2. **タスク登録画面**: 新規タスク作成
3. **タスク詳細・編集画面**: 既存タスクの編集・削除
